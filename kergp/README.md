# Codes based on kergp R package

Our R implementations are based on the kergp package (kergp==0.5.0)

### Dependencies:
1. kergp (kergp==0.5.0, modified): the source files for installing the package are available in ./kergp_installer/
2. Other dependecies are automatically added by installing kergp.

Multi-output Gaussian processes with functional data: A study on coastal flood hazard assessment.
